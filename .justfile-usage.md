## Installation

```bash
# macOS
brew install just

# Linux (cargo)
cargo install just

# Or download from https://github.com/casey/just
```

## Quick Start

```bash
# See all commands
just --list

# Get help
just help

# Check current status
just status

# Validate Phase 1 foundation
just phase-1
```

## Example Workflows

### Start Development Environment
```bash
just dev-start
# Deploys: k0s cluster + Postgres + Redis + Agents
```

### Work on Customer Module
```bash
# Execute checkpoint
just checkpoint-2-1

# Implement changes in crates/customer/

# Validate and commit
just commit-checkpoint "2.1" "Customer module foundation"
```

### Infrastructure Management
```bash
# Deploy
just infra-deploy

# Check status
just infra-status

# Apply migrations
just db-migrate

# Verify
just db-verify
```

### Multi-Agent Hive
```bash
# Start agents
just agents-start

# Monitor
just agents-logs

# Check results
just agents-results
```

### Development Cycle
```bash
# Full cycle: format, lint, test, build
just dev

# Watch mode
just watch

# Run tests with coverage
just test-coverage
```

### Checkpoint Management
```bash
# Save current work
just commit-checkpoint "2.2" "Customer authentication"

# Soft backtrack (keep changes)
just backtrack

# Hard reset (lose changes)
just backtrack-hard

# View history
just checkpoint-history
```

## Command Categories

| Category | Commands | Purpose |
|----------|----------|---------|
| **Phases** | `phase-1`, `phase-2`, etc. | Execute full phases |
| **Checkpoints** | `checkpoint-2-1`, etc. | Individual checkpoints |
| **Infrastructure** | `infra-deploy`, `infra-status` | k0s cluster management |
| **Database** | `db-migrate`, `db-verify`, `db-shell` | Postgres operations |
| **Agents** | `agents-start`, `agents-logs` | Multi-agent hive |
| **Build** | `build`, `test`, `lint`, `fmt` | Cargo operations |
| **Dev** | `dev`, `watch`, `validate` | Development workflow |
| **Utility** | `status`, `clean`, `help` | Helpers |
